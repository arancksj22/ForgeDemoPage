Forge

Oct 2025 - Nov 2025

Spring Boot, RabbitMQ, Docker, AWS S3, PostgreSQL, Angular

A cloud-native build automation platform that spins up isolated Docker containers on-demand, supporting 10+ concurrent builds across 4 language ecosystems with automated dependency installation and artifact management.

Key Features

  • Developed Forge, a cloud-native build automation platform using Spring Boot and Angular that spins up isolated Docker containers on-demand, supporting 10+ concurrent builds and eliminating environment configuration drift across development teams.
  • Implemented asynchronous job processing with RabbitMQ to queue build requests, decoupling API responses from long-running builds and enabling 3 concurrent worker processes.
  • Automated build execution using Testcontainers and the Docker SDK to provision short-lived containers, install dependencies, run builds, and extract artifacts in under 2 minutes per build across 4 language ecosystems (Java, JavaScript, Python, Kotlin).
  • Configured AWS S3 with presigned URLs for secure artifact upload/download, storing 75+ build outputs totaling 4 GB with automatic expiration policies, and Supabase (PostgreSQL) to track job metadata.
